About

Hey there, I'm Lisandro!

I am a developer and university student of engineering in information systems.
I enjoy writing code to create new projects and provide solutions. I like to improve myself to continue growing and incorporating new skills.

My interest in development began when I was surfing the Internet and I found it interesting to be able to change the appearance of a web page, such as color, text and images. So, I looked into how this was possible and started modifying the pages through the development tools included in web browsers.

Then I started studying information systems engineering, where I learned development languages like Java and Python and started developing software applications, at the same time I learned to create solutions in a more optimal way following development patterns. I also learned about network and communications technologies, databases, operating systems, and computer architecture.

I am currently taking a web development course to continue training and learning new skills in Java, JavaScript, TypeScript, BootStrap, SpringBoot, NodeJS and Angular.
